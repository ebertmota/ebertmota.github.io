Linux

Desinstalação de Progamas

Para verificar os progamas que você tem instalado em um linux baseado em debian basta digitar "sudo dpkg -l".

Para salvar a sua lista de progamas instalados em um arquivo de texto use os comandos: cd ~ e sudo dpkg - l > lista.txt

* Para desinstalar o progama e manter suas configurações salvas use: sudo apt-get autoremove nome_do_progama

* Para desisntalar completamente sudo apt-get purge nome_do_progama
