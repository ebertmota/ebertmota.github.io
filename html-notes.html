HTML

* Um documento HTML Válido precisa seguir obrigatoriamente a estrutura composta pelas tags <html>, <head>, <body> e a instrução <!DOCTYPE>.

* Para escrever em seu texto as tags html use &+lt para "<" e &+gt para ">" (+ apenas representativo).

A tag <iframe>...</iframe> é usada para que seja inserido o vídeo dentro do iFrame.

O iFrame source (src) é a origem do conteúdo, seja do servidor interno ou externo.

Width e Height são aspectos visuais do iFrame. Você pode inserir uma medida fixa em Pixels ou em porcentagem.

Para colocar qualquer video do Youtube por exemplo em seu site, abra o video no proprio youtube e em compartilhar selecione Incorporar e receberá o codigo para adiciona lo em seu Html.

Label

Label = Rótulo,Legenda ou Etiqueta

A tag <label> define um Rótulo para um elemento <button>, <input>, <meter>, <progress>, <select> ou <textarea>

O Elemento Label nao é renderizado como algo especial para o user. Contudo fornece uma melhoria de usabilidade para user com mouse, porque se o user clicar no texto dentro do elemento <label>,ele alterna o controle.

O atributo for da tag <label> deve ser igual ao id do elemento relacionado para eles serem vinculados.

Para vincular a Label ao elemento desejado use o atributo for ou insira a <label> dentro do elemento.

Atributos

acesskey

A tecla de atalho para acessar este elemento a partir do teclado.

for

O ID de um elemento de formulário relacionado, ou seja o primeiro elemento no html com ID de mesmo valor a esta propriedade.

form

Este atributo permite que você coloque elementos de label em qualquer lugar dentro de um documento, e não apenas como descendentes de seus elementos de formulário.

Span

Um elemento <span> pode ser usado para colorir uma parte de um texto.

A tag é usada para agrupar elementos inline em um documento.

Fornece a maneira de adicionar um vinculo a uma parte de texto ou elementoo.

Quando um texto é vinculado a um elemento %ltspan>, você pode estilizá-lo com CSS ou manipulá-lo com JavaScript.
