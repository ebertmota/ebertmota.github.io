HTML

* Um documento HTML Válido precisa seguir obrigatoriamente a estrutura composta pelas tags <html>, <head>, <body> e a instrução <!DOCTYPE>.

* Para escrever em seu texto as tags html use &+lt para "<" e &+gt para ">" (+ apenas representativo).


iFrame

É um componente de um elemento HTML que permite incorporar documentos, vídeos e mídia interativa em uma página, basicamente permite incorporar elementos de outros sites no seu Html. Um exemplo de iframe é o video abaixo:

A tag <iframe>...</iframe> é usada para que seja inserido o vídeo dentro do iFrame.

O iFrame source (src) é a origem do conteúdo, seja do servidor interno ou externo.

Width e Height são aspectos visuais do iFrame. Você pode inserir uma medida fixa em Pixels ou em porcentagem.

Para colocar qualquer video do Youtube por exemplo em seu site, abra o video no proprio youtube e em compartilhar selecione Incorporar e receberá o codigo para adiciona lo em seu Html.
