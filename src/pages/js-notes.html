Arrow Function

As Arrow Functions foram introduzidas no ES6.

e nos permite escrever uma função com uma sintax mais curta.

Exemplo de função sem sintex Arrow:

hello = function() {

return "Hello World";

};

Exemplo com Arrow Function:

hello = () => {

return "Hello World";

};

Se a função tiver e receber somente uma instrução ela pode ficar ainda mais curta:

hello = () => "Hello World";

Arrow Function com parâmetros

hello = (val) => "Hello World" + val ;

Se voce tiver somente um parâmetro pode tirar os parênteses:

hello = val => "Hello World" + val ;

Tratamento de Dados

o + pode servir tanto para adição ou concatenação.

number + number => adição

string + string => concatenação

Para converter uma string em número:

Number.parseInt(n) // conversão para inteiro

Number.parseFloat(n) // conversão para float(flutuante)

Number(n) // converte para numero mas deixa o js decidir se vai ser int ou float

Para converter de número para string:

String(n)
