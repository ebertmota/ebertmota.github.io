A propriedade controla o tamanho da viewport. O valor desta propriedade pode ser definido com um número específico de píxels como por exemplo ou com um valor especial chamado que representa a largura da onde o conteúdo está sendo apresentado em pixels de CSS considerando uma escala de 100%. (Ainda há as propriedades e , as quais podem ser úteis para páginas com elementos que mudam de posição baseado na altura da viewport.)

A propriedade controla o nível de amplificação quando a página é carregada pela primeira vez. As propriedades , , e controlam a permissão para o usuário efetuar aumento ou diminuição da página.

Alem disso a meta viewport auxilia para o funcionamento de media query nos browsers.

Display

Css syntax: display: value;

JavaScript syntax: object.style.display="none"

A propriedade Display especifica o comportamento do display

Propriedades e valores

Valor Descrição
inline Seta o display para um elemento de uma linha como <span>, Qualquer propriedade height ou width não irão fazer efeito.
block Seta o display para um elemento de bloco como o <p>, Começa sempre em uma nova linha e tem width de 100%.
contents Faz o container desaparecer, transformando os elementos filho (child) dele, como filhos(children) do elemento no proximo nivel do DOM.
flex Define o display do elemento como block-level flex container, ou seja flexivel
grid Define o display do elemento como block-level grid container
inline-block Define o display para um elemento inline-level block container. O elemento por si é formado do mesmo jeito de um elemento inline, mas você pode aplicar valores a height e width.
inline-flex Elemento como inline-level flex container
inline-grid Elemento como inline-level grid container
inline-table O elemento é configurado como inline-level table
list-item Seleciona o comportamento do elemento para como o do elemento <li>
run-in Seleciona o display para ou block ou inline, dependendo do contexto
table Define o mesmo comportamento do elemento <table>
table-caption Define o mesmo comportamento de um elemento <caption>
table-column-group Define o mesmo comportamento de um elemento <colgroup>
table-header-group Define o mesmo comportamento de um elemento <thead>
table-footer-group Define o mesmo comportamento de um elemento <tfoot>
table-row-group Mesmo comportamento de um elemento <tbody>/td>
table-cell Mesmo comportamento de um elemento <td>
table-column Mesmo comportamento de um elemento <col>
table-row Mesmo comportamento de um elemento <tr>
none O Elemento é completamente removido da tela
initial Seleciona a propriedade do display para o seu valor padrão
inherit Herda essa propriedade do seu elemento pai
